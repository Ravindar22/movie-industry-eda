# Movie Industry Exploratory Data Analysis

## Overview
This project focuses on exploratory data analysis (EDA) of a large movie dataset to understand trends and patterns in the movie industry. The analysis explores relationships between genres, release years, budgets, box office performance, and audience ratings.

## Dataset
- Source: Kaggle (Synthetic Movie Dataset for Analytics and Visualization)
- Size: ~1 million movie records
- Time Period: 1950 to 2025
- Data Type: Synthetic but statistically realistic

## Objectives
- Understand distribution of movies across genres and years
- Analyze budget and box office revenue trends
- Explore relationships between financial performance and audience ratings
- Gain hands-on experience with large datasets and EDA workflows

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory analysis of numerical and categorical features
- Visualization of trends and relationships
- Insight extraction from observed patterns

## Conclusion
This project demonstrates practical experience in exploratory data analysis and data visualization using Python. The insights derived can support further analysis, dashboarding, or predictive modeling in future work.
